# Supply Chain Optimization Tool

A Streamlit MVP that walks through demand forecasting, inventory planning, and simulation to explain supply chain tradeoffs.

## Quickstart
### Prerequisites
- Python 3.11+

### Run locally
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Use `demo_demand.csv` as a sample input.

## Tests
```bash
python -m unittest discover -s tests
```
